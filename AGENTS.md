## Project Details

Terranaut is an expo mobile application (`mobile/`) and react website (`web/`) powered by a go lang api backend (`backend/`) where users have fun being more active and getting outside.

## Development

- After making substantial code changes, always run `pnpm run lint:agent`. This will automatically fix fixable linter issues and will display issues you must fix.
- Once a feature is complete, verify your changes work by running `pnpm run build`.
